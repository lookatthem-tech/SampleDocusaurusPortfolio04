  <h2>Audience</h2>
  <p>
    This Guide is intended for Prysm trained and qualified field engineers,
    partners and, and customer Network Administrators. This guide presumes that
    the audience has a full understanding of basic network provisioning and
    common Microsoft user authentication methods.
  </p>
  <h2>Safety Precautions</h2>
  <p>
    Only qualified service technicians should install and configure the
    Application Appliance. Basic electrical safety precautions should be
    followed.
  </p>
  <h2>Warnings</h2>
  <p>To ensure that the Application Appliance operates without error:</p>
  <ul>
    <li>
      Do not install third-party software on a Prysm Application Appliance
      unless specifically required for new features or functionality as
      specified in a Purchase Order request.
    </li>
    <li>
      Do not update Windows OS device drivers on the Prysm Application
      Appliance.
    </li>
    <li>
      Do not enable Windows update or updates for any installed software on the
      Prysm Application Appliance.
    </li>
    <li>
      Only use the applications and configuration files located in the&nbsp;
      <em>C:\Tools</em>&nbsp;directory in accordance with this and
      other&nbsp;Prysm documents.
      <br />
    </li>
  </ul>
