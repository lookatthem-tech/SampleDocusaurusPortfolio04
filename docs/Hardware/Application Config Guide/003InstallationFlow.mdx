# Installation Flowchart
  <p>
    The following flowchart, for installing and configuring a Prysm Application
    Appliance, provides a visual overview of the subsequent sections described
    herein.
  </p>
  <p>
    The flowchart refers to the four C's of installation and configuration with
    the corresponding Section numbers of this Guide:
  </p>
  <p style={{ marginLeft: "30.0px" }}>
    <strong>2. Connection&nbsp;</strong>of&nbsp;the Prysm Application Appliance.
  </p>

    Appliance.
  </p>

    <strong>4. Customization and Options</strong>.
  </p>
  <p style={{ marginLeft: "30.0px" }}>
    <strong>5. Confirmation of Installation</strong>&nbsp;and verification of
    customized functionality.
  </p>
  <p>
    <strong>Assumptions:</strong>
  </p>
  <ul>
    <li>An Organizational Account exists within the Prysm Web Admin</li>
    <li>
      A Deployment has been created in the Prysm Web Admin that associates the
      MachineKey with a Settings Profile
    </li>
    <li>
      The installer (FAE) has been trained and provided sufficient permissions
      within the Prysm Web Admin to apply configuration&nbsp;changes for the
      specified account
    </li>
