# Prerequisites and Installation Guide Flowcharts

<p>
The Prysm Application Appliance and Prysm Application
Suite software are configured to default factory
settings. Additional field configurations are
necessary to reflect the integration and functional
requirements for each customer. Before configuring the
Application Appliance, review this section and gather
all necessary materials.
</p>
<h2>Prerequisites</h2>
<p>
Field installation and support personnel must be
familiar with the following applications and
procedures:
</p>
<ul>
<li>
Prysm Digital Workplace and touch sensor
installation and operation
</li>
<li>
Microsoft Windows 8.1 and Windows 10 desktops,
applications, directory structure, navigation, text
editing through Notepad{" "}
</li>
<li>
Nvidia control panel for navigating display
resolutions settings
</li>
<li>
Basic Prysm Application Suite end user functions
</li>
<li>
Salesforce access for viewing and editing the
Application Appliance Installation Record
</li>
<li>
VLC, VNC, and Windows network configuration process
</li>
<li>VC or Video Conferencing Integration</li>
<li>
Third-Party Applications Integration
<br />
<ul>
<li>File Storage Provider Integration</li>

</ul>
